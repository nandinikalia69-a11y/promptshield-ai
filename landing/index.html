🔧 How It Works

Three Layers of Protection

Layer 1
🔍 Detection Engine

Modular regex-based detectors scan prompts for 20+ sensitive data patterns across 6 categories.

  • EmailDetector — RFC 5322 patterns
  • APIKeyDetector — 14 key formats
  • FinancialDetector — PCI-DSS patterns
  • SourceCodeDetector — 6 languages
  • CredentialDetector — Password patterns
→ { threats[], categories[], count }
Layer 2
⚖️ Risk Engine

Composite risk score using weighted severity model. Considers threat count, categories, and diversity.

  • CRITICAL: 35 points/threat
  • HIGH: 20 points/threat
  • MEDIUM: 10 points/threat
  • LOW: 3 points/threat
→ { score: 0–100, level, explanation }
Layer 3
✂️ Redaction Engine

Ordered pattern substitution with context-aware redaction. Preserves prompt structure and meaning.

  • Priority ordering (critical first)
  • Smart phone disambiguation
  • Structural preservation
  • Diff generation for preview
→ { sanitized, redactionCount, diff }
